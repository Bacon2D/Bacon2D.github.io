    Detailed Description

    The Game component is the root game component which contains one or more Scene components. Only one Scene will be set as currentScene.

    Example usage:

    import QtQuick 2.0
import Bacon2D 1.0

Game {
    id: game
    width: 800
    height: 600

    Scene {
        id: scene
        width: 300
        height: 300
    }
}
